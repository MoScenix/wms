# WMS

[![Go](https://img.shields.io/badge/go-1.25-blue.svg?style=flat-square)](https://go.dev/dl/)

> 基于 **Kitex** 和 **Hertz** 的分布式仓储管理系统（WMS）

## 概览

+ 基于 Docker，支持一键启动基础依赖
+ 前后端分离，BFF + RPC 微服务架构
+ 登录态 + 角色权限，覆盖仓库核心业务流程
